# Retail Sales Analysis using MySQL

## Project Overview

This project analyzes a retail sales dataset using MySQL to generate business insights. The objective is to answer real-world business questions related to sales performance, customer behavior, regional performance, and product analysis using SQL.

## Tools Used

- MySQL 8.0
- Visual Studio Code
- GitHub
- Microsoft Excel

## Dataset

The dataset contains 100 retail sales transactions with information such as:

- Customer Details
- Product Details
- Sales Amount
- Profit
- Payment Method
- Region
- Delivery Status

## SQL Concepts Used

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()
- DISTINCT
- LIKE
- BETWEEN
- IN
- HAVING
- CASE
- LIMIT

## Business Questions Solved

This project answers 30 real-world business questions, including:

- Total Orders
- Total Sales
- Total Profit
- Average Order Value
- Sales by Product Category
- Profit by Region
- Top Customers by Sales
- Payment Method Analysis
- Most Profitable Orders
- Customer and Product Insights
